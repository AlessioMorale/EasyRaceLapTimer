# EasyRaceLapTimer IR_DAEMON

## requirements

    sudo apt-get install  qt5-default libcurl4-openssl-dev

## compilation

    cd ir_daemon
    qmake ir_daemon.pro
    make
    sudo ./ir_daemon [for starting]
